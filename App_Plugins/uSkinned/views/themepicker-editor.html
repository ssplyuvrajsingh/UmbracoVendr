An issue has occured getting your frontend source files.

Please follow these steps to make sure that your uSkinned theme is configured correctly.

1. Check the following file exists: ~/config/usntheme.config

2. Check this file contains at least the following:

<USNThemeConfig>
  <themes>
    <add name="uSkinned" />
  </themes>
</USNThemeConfig>

3. If adding your own customisations to the base code it is recommended that you duplicate the uSkinned folder within ~/css, ~/scripts, ~/views/partials with your own folder name and add your folder name as shown below.

<USNThemeConfig>
  <themes>
    <add name="uSkinned" />
    <add name="yourfoldername" />
  </themes>
</USNThemeConfig>

4. Check ~/web.config contains the following:

<configuration>
  <configSections>
    <section name="USNThemeConfig" type="USNStarterKit.USNHelpers.USNThemesConfigsSection" />
  <configSections/>
  <USNThemeConfig configSource="config\usntheme.config" />
<configuration/>
